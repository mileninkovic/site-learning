Data types

JavaScript has, as most programming languages, several data types. Some of the different data types are described below.

Number

A number in JavaScript is defined without the use of quotes or other symbols when declaring a variable. var x = 3; is an example of a number data type.

String

A string in JavaScript is defined by using a double quote ", a single quote ' or a backtick ` when declaring a variable. var x = "string"; is an example of a string data type.

Boolean

A boolean is either true or false. In JavaScript it is defined without the use of any quotes or other symbols when declaring a variable. An example of a boolean could be var x = false;
