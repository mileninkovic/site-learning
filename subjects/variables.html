Variables

In JavaScript there are three different keywords that declare variables. These are var, let and const. These keywords are followed by a variable name, an equal sign and lastly a value. Below is an example of different variables.

            var example1 = 0;
let example2 = 0;
const example3 = 0;

var

var has global scope when outside of a function block. If var is inside a function block it has local scope. We can also redeclare a var as long as it is in the same scope.

let

let is the preferred keyword to use when declaring variables. let is block scoped, when declared it can only be used in the block it is declared in. let can be updated but not re-declared.

const

const is used for declaring constant values or values that don't change. const values are block scoped just like let. It is not possible to update or re-declare const values.
